# 🎯 Student Grade Calculator

## 📌 Project Overview
The Student Grade Calculator is a Python program that takes a student's name and marks as input and calculates their grade based on predefined rules. It also provides encouraging messages based on performance.

## 🎯 Objectives
- Learn Python basics (input/output, functions)
- Understand conditional statements (if-elif-else)
- Implement input validation using loops
- Create reusable functions

## ⚙️ Setup Instructions
1. Install Python (version 3.x)
2. Open terminal or command prompt
3. Navigate to project folder
4. Run the program:
   python grade_calculator.py

## 📂 Code Structure
- `calculate_grade()` → Function to determine grade and message
- `while loop` → Handles invalid inputs
- `if-elif-else` → Implements grading logic

## 🧠 Grading Logic
- A: 90–100 → Excellent
- B: 80–89 → Very Good
- C: 70–79 → Good
- D: 60–69 → Needs Improvement
- F: 0–59 → Fail

## 🖥️ Features
- Input validation (0–100 only)
- Error handling for invalid inputs
- Encouraging messages for each grade
- Clean and user-friendly output

## 🧪 Testing
Multiple test cases were used:
- Valid inputs (e.g., 85 → Grade B)
- Invalid inputs (e.g., -10, 200, text)
- Edge cases (0 and 100)
